## 🐛 XỬ LÝ SỰ CỐ

### Nếu server không chạy:
```bash
cd backend
npm install
npm start
```

### Nếu port 3000 bị chiếm:
**Windows:**
```powershell
netstat -ano | findstr :3000
taskkill /PID [số_PID] /F
```

### Nếu không đăng nhập được:
- Clear browser cache: Ctrl + Shift + Delete
- Hard refresh: Ctrl + Shift + R
- Check console: F12 → Console tab

### Nếu giao diện không đúng:
- Đảm bảo truy cập http://localhost:3000 (không phải các port khác)
- Kiểm tra file index-v2.html đã load đúng styles-v2.css và app-v2.js chưa
