## 🔄 CÁCH CHUYỂN SANG V2:

### Bước 1: Cài dependencies mới
```bash
cd backend
npm install
```

### Bước 2: Đổi file server
Trong `backend/package.json`, đổi:
```json
"start": "node server-v2.js"
```

### Bước 3: Test local
```bash
npm start
```
Mở http://localhost:3000

### Bước 4: Push lên GitHub
```bash
git add .
git commit -m "Upgrade to v2.0: Professional UI with Auth"
git push
```

### Bước 5: Redeploy trên Render
Render sẽ tự động detect và redeploy!
